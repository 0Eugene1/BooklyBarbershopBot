Telegram-бот для онлайн-записи в барбершопы.

## О проекте

BooklyBarbershopBot позволяет пользователям быстро записываться на услуги барбершопов через Telegram. Проект ориентирован на удобство: выбор филиала через уникальный `slug`, интеграция с Yclients API, хранение истории бронирований и возможность отмены записи.  

Бот также обеспечивает безопасный сбор контактной информации и взаимодействие с пользователями через inline-кнопки.

## Основной функционал

- `/start` — начать взаимодействие с ботом:
  - с указанием `slug` конкретного барбершопа;
  - без `slug`, с предложением выбрать филиал через официальный Telegram-канал.
- **Выбор услуг мастеров** с отображением стоимости.
- **Выбор даты и времени записи** через интеграцию с Yclients API.
- **Подтверждение номера телефона** через Telegram.
- **Отмена записи** и хранение `recordId` для корректного взаимодействия с API.
- **Сохранение последнего использованного филиала** (`lastUsedSlug`) для быстрого повторного доступа.
- **Интеграция с официальным Telegram-каналом** для удобного выбора филиала.
- Inline-кнопки для удобного взаимодействия пользователя с ботом.

## Архитектура

- **Spring Boot** — основной фреймворк для бэкенда.
- **TelegramBots API** — для взаимодействия с Telegram.
- **Yclients API** — интеграция с системой бронирования барбершопа.
- **PostgreSQL** — база данных для хранения клиентов, бронирований и состояния.
- **WebClient / REST** — асинхронные запросы к API.
- **DTO, Service, Repository слои** — чистая архитектура для удобства поддержки и расширения функционала.
